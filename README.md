行业侦察一体化引擎——双层闸门验证体系：前置闸门（铁律0+铁律1）逐条检验不通过不准动笔，最终闸门（铁律3）全量铁律逐条检验不通过不准交付。融合罗盘行业研究引擎核心能力：权力分析框架、多报告交叉对比、金字塔原则标题、政策权重强化、5类行业视角权重表、HTML可视化输出。
支持三种行业类型路由：硬科技（产业链拆解为主）、消费品（品牌/渠道/用户为主）、服务/平台（商业模式/网络效应为主）。
第一层「行业判断框架」：生命周期定位 + 四维分析 + 估值 + PEST + 景气度。
第二层「行业侦察实操」：硬科技走产业链拆解（正逆向闭环追问 + 话语权分析）、消费品走品牌渠道用户、服务/平台走护城河+单位经济。
独立专项模块：品牌与商品五段分析 | 行业对比8维矩阵 | 投资机构26家数据库+五维评分。
∞无限制模式：先完整深度分析框架，再五步深化分析（背景→现状→推演→正反面案例→综合交叉验证），按框架顺序融合输出。
两条分析铁律贯穿全流程（多元视角+连锁效应），双层闸门验证体系（前置闸门铁律0+1不通过不准动笔，最终闸门铁律3逐条全检不通过不准交付）。
