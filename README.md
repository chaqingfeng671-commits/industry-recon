🔭 行业侦察·天眼

行业侦察一体化引擎 — 输入一个行业名称，自动完成从行业定位、产业链拆解、竞争格局、投资判断的全流程分析。适合创业调研、投资研究、竞品分析、行业入门。

核心能力

天眼不是固定模板，而是按行业类型自适应路由的分析引擎：硬科技 / B2B 制造走产业链拆解（正逆向闭环 + 话语权分析），消费品 / B2C 走品牌矩阵 + 渠道分销 + 用户画像，服务 / 平台走商业模式拆解 + 护城河 + 单位经济。

全链路分析框架分三层： 第一层行业判断框架（生命周期定位 + 四维分析 + PEST + 景气度，政策权重优先）；第二层行业侦察（按路由分支执行，硬科技每层标注话语权评分，多报告交叉比对框架差异和视角矛盾）；第三层裁断与验证（交叉验证 + 全息报告生成，交付前双层闸门逐条检验）。

独立专项模块： 品牌与商品五段分析（一个商品为什么火的完整拆解）、行业对比 8 维矩阵（双行业独立分析 + 横向对比）、投资机构 26 家数据库（查询机构信息 + 被投公司五维评分 + 前景预测）。

分析模式

快速判断	XX行业怎么样	生命周期定位 + 四维评分 + 一句话结论	5-10 min

标准侦察	侦察XX行业	全息报告（.md + .html）	20-40 min

深度分析	深度分析XX行业	完整产业链 + 技术路线 + 竞争格局	子 session 并行

专项拆解	拆解XX竞品/XX产业链	单一方向深度拆解	10-15 min

∞ 无限制	往死里挖XX行业	深度分析 → 背景溯源 → 推演预测 → 正反面案例 → 最终交叉验证	子 session 并行

∞ 无限制模式分两阶段：第一阶段深度分析框架完整走一遍（路由 → 判断 → 侦察 → 裁断），第二阶段深化分析（背景归因 → 现状深化 → 推演预测1/3/5年三种情景 → 正反面案例五维对比 → 最终综合交叉验证），全部结论标注置信度变化。

强制铁律

天眼底层是三条强制执行铁律（0/1/2）和两条贯穿全流程的分析铁律（A/B），双层闸门确保执行质量。执行流程：铁律0+1 → [前置闸门：逐条检验，不通过退回] → 铁律2 → 铁律A+B → [最终闸门：全量逐检] → 交付。

铁律0 — 环境三检自检： 任何分析前强制检查技能加载、沙箱环境、联网搜索，三项通过才进入流程。铁律1 — 先想后搜： 没想清楚不准动笔，强制拆解问题、审视流程、规划搜索和篇章结构。铁律2 — 多模块独立执行： 每个模块独立搜索分析产出，A模块没数据标注缺口，B模块照常跑。铁律A — 多元视角： 每条核心判断从≥3个视角（投资者/经营者/技术演进者/监管者/客户）分别论证，视角冲突要明写。铁律B — 连锁效应： 每个关键变化信号展开连锁传导（触发事件→第一层→第二层→第三层扩散，含反向传导、利益重分配、时间差）。

前置闸门（动笔前）： 逐条检验铁律0（3项）+ 铁律1（9项），任何一项❌退回对应步骤修正。最终闸门（交付前）： 逐条验证每条铁律的每条子项，标准侦察及以上全部逐检，退回只退未达标子项对应步骤。最终检验信号：报告读完后能感受到"这个行业既在爆发也在流血"的张力。

输出形式

快速判断输出 .md，标准侦察及以上输出 .md + .html（卡片式公司清单、三色风险标注、来源标签），深度/∞模式子 session 并行执行后汇总报告。所有报告标题采用金字塔原则：判断句，结论先行。

边界说明

只分析上市公司与知名公司（财报支撑话语权指标计算），不做具体投资回报率承诺，不做未公开数据编造（搜不到就标注缺口），所有结论基于公开信息与AI判断，存在信息滞后和认知偏差。不替代专业法律/医疗/财务咨询。

使用方式

直接说行业名即可触发：侦察储能行业、深度分析光通讯、拆解新能源汽车产业链、消费品行业分析看看宠物食品、往死里挖商业航天、对比半导体和光伏行业。
